--- Current Features ---<br>
CheaterCog:<br>
 >!CheckCheater - lookup tool to see if someone is a known cheater<br>
 >!ListCheaters - lists all known cheaters<br>
 >!LogCheater - adds new cheater to the list<br>
 
ClassSetupCog:<br>
 >!ClassSetup - placeholder<br>
 
ModerationCog:<br>
 >!Warn - warns user<br>
 >!Mute - mutes user<br>
 >!Ban - bans user<br>
 >!Unmute - unmutes user<br>
 >!Unban - unbans user<br>
 >!Remove - removes specified warning<br>
 
ModmailCog:<br>
 >!CloseTicket - closes modmail ticket<br>
 
ReactionRolesCog:<br>
 >!AddCategory - adds a reaction role category<br>
 >!AddReactionRole - adds a reaction role to an existing message<br>
 
WelcomeCog:<br>
 >!WelcomeRoles - adds roles given upon joining a server<br>
